STEP 1: Creating a Simple Tool

def calculator(math_problem):
    result = eval(math_problem)
    return f"Answer: {result}"

We define a basic calculator tool using Python's eval() function.

STEP 2: Connecting to OpenAI

from langchain_openai import ChatOpenAI
from dotenv import load_dotenv

load_dotenv()
llm = ChatOpenAI()

response = llm.invoke("What is the capital of France?")
print(response.content)

We load our API key securely and test if the LLM is connected by asking it a simple question.

STEP 3: Defining AI Tools

from langchain.agents import Tool

tools = [
    Tool(
        name="Calculator",
        func=calculator,
        description="Useful for simple math like '2 + 2'"
    )
]

We define what tools our agent will have access to â€” in this case, just the calculator.

STEP 4: Creating the Agent

from langchain.agents import initialize_agent, AgentType

agent = initialize_agent(
    tools=tools,
    llm=llm,
    agent=AgentType.ZERO_SHOT_REACT_DESCRIPTION,
    verbose=True
)

This sets up our LangChain agent with reasoning capability based on tool descriptions.

STEP 5: Testing the Agent

agent.run("What is 23 * 17?")

The agent sees that itâ€™s a math problem and uses the calculator tool!
